# 1. Önce projedeki TÜM dosyaları istatistikten düşürüyoruz (Temizlik)
* linguist-vendored

# 2. Sadece Python dosyalarını "Bunlar benim kodum" diyerek geri açıyoruz
*.py linguist-vendored=false
*.py linguist-detectable=true
*.py linguist-language=Python

# 3. Markdown (README) dosyaları da görünsün
*.md linguist-vendored=false

# 4. TeX, HTML, CSS gibi rapor dosyalarını özellikle engelliyoruz (Garanti olsun)
*.tex linguist-vendored
*.html linguist-vendored
*.css linguist-vendored
*.ui linguist-vendored
